规则
描述：
所有控制器文件必须存放至 controllers 目录下，框架根据路径自行判断使用哪个控制器。
路径示例： 
http://www.xxx.com/admin/login
解说： 
程序会首先判断 controllers 目录下是否存在 Admin（首字母会变成大写） 目录，若存在则会加载 controllers/Admin/Controller.php 类，同时加载 controllers/Admin/Login.php 类，此时使用的是 Login 控制器。
这时 Controller.php 中必须定义名为 Controller 的类，并集成自 Main 类；Login.php 中必须定义名为 Login 的类，并继承 Controller 的类。
若 Admin 目录不存在，则直接加载 controllers/Login.php 类，此时使用的是 Login 控制器。
加载的类文件： 
controllers/Admin.php (如果 Admin 是个文件夹)
controllers/Admin/Controller.php controllers/Admin/Login.php (否则加载 Admin 目录下的 Controller 以及 Login)

变量
列表： 
array param，代表网址中的参数数组序列
string action 代表控制器中的方法部分
示例： 
URL: /admin/login/shop/go
此时若 admin 不为控制器下的目录，则：
加载 Admin.php，login 为 Admin 类的方法，且 action 值为 login
shop/go 会成为 param 的值，param[0] 为 shop，param[1] 为 go，以此类推
-
若 admin 为控制器下的名录，则：
加载 Admin/Login.php，shop 为 Login 类的方法，且 action 值为 shop
go 会成为 param 的值，param[0] 为 go，以此类推
获取： 
在控制器中您可以直接使用 $this->action 和 $this->param 获取变量。
