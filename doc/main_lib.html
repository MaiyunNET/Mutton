加载
描述：
一句代码加载您编写的，或者第三方提供的类库。
定义： 
L()->load(string $name)
参数 描述 返回值
name 直接传入类库的名字，允许带目录，不带 L_ 前缀，并自动初始化 void 
L()->load(string $name, bool $autoinit)
参数 描述 返回值
name 同上，但根据 autoinit 的值决定是否自动初始化 void
autoinit 为 true 则自动初始化，为 false 不进行初始化
例子： 
L()->load('Sql'); // Sql 类被装载，并且已经自动初始化
L()->load('Mysql', false); // Mysql 类被装载，但没有进行初始化
$mysql = new L_Mysql(); // 手动初始化

小提示：手动初始化可能应用在当前页面需要使用同个类多实例，譬如多条同时存在的数据库连接。

使用
描述：
若是您自己实例化的，直接那么用就行，初始化的适用本条。
定义： 
L()->类名->方法();
例子： 
L()->load('Sql');
L()->Sql->get(); // get 是 Sql 类的一个方法
