Memcached 类
本类是 1.0 新增的。
简介 此类在于创建 Memcached 连接并进行操作，支持连接池，支持 SASL。
设定 预定义 构造
方法 on

预定义
描述： 
使用 Memcached 您需要进行预定义一些常量，通常放置在 confs/db.php 文件当中。
当然，您也可以不定义常量，而在每个方法需要使用常量的地方自行传入参数即可。
列表：
类型 常量 描述
string MCHOST 服务器名
string MCPRE 存储的 key 前缀
int MCPORT 服务器端口号
string MCUSER SASL 用户名
string MCPW SASL 密码
string | NULL MCPOOL 连接池别名，为 NULL 则不使用连接池
例子： 
define('MCHOST', '127.0.0.1');
define('MCPRE', 'pre_');
define('MCPORT', 11211);
define('MCUSER', 'user');
define('MCPW', '123456');
define('MCPOOL', 'BSL');
