全局常量
ROOT_PATH 
程序的根路径，用于在服务端包含 PHP 等文件时使用。
ROOT_PATH 的内容形如 D:\www\htdocs\maiyun.net\www\
LIB_PATH 
LIB_PATH 相当于 ROOT_PATH + 'librarys/'
MOD_PATH 
MOD_PATH 相当于 ROOT_PATH + 'models/'
CONTROL_PATH 
CONTROL_PATH 相当于 ROOT_PATH + 'controllers/'
AJAX 
若本次请求来自 Ajax，则此变量为 true，否则为 false。
通常这么使用 if(AJAX) {} else {};
SITE_PATH 
对于客户来说程序的网站根，通常用于引入 JS、CSS 等静态文件，你完全无需考虑你的网站是在根还是二级目录，SITE_PATH 都能准确识别。
如果根目录就是网站所在目录，那么 SITE_PATH 则为 /，如果网站不再根目录，在多级目录下，SITE_PATH 则类似为 /website/maiyun.net/
引用 JS 形如：<script src="<?php echo SITE_PATH ?>images/main.js"></script>
使用可更新缓存常量则形如 <script src="<?php echo SITE_PATH ?>images/main.js?<?php echo IMAGE_VER ?>"></script>
IMAGES_PATH 
性质和 SITE_PATH 类似，唯一不同的是比 SITE_PATH 多了一个 images/，相当于 IMAGES_PATH = SITE_PATH + "images/"，用于快速的构建引入 JS 等静态文件的前缀路径。
IMAGES_PATH 的值形如 /images/、/websit/maiyun.net/images/
HTTP_PATH 
性质和 SITE_PATH 类似，唯一不同的是比 SITE_PATH 多了前缀域名，相当于 HTTP_PATH = "http://www.xxx.com" + SITE_PATH，主要用于传递回调、跳转之用。
HTTP_PATH 的值形如 http://www.maiyun.net/、http://127.0.0.1/website/maiyun.net/
PAGE_START_TIME 
页面开始的时间，有很长的小数点，通常用来计算页面总执行时间所用。
PAGE_START_TIME 的值形如 1436353310.7302629947662
注意：受限于 PHP 对数字的显示原因，若您直接输出的话小数点可能只有 4 位，但实际上其他位可以参与运算。
